# 🚀 Projeto: Portfólio Profissional em Docker

Este repositório contém a configuração de infraestrutura e o código-fonte do meu currículo profissional, desenvolvido para consolidar conhecimentos em conteinerização, administração de servidores Linux e práticas de DevOps.

## 🛠️ Tecnologias e Ferramentas

* **Docker & Docker Compose**: Orquestração do ambiente e isolamento de dependências.
* **Nginx (Alpine)**: Servidor web de alta performance utilizado para servir os arquivos estáticos.
* **Samba**: Configurado para integração e transferência de arquivos entre o host (Windows) e a VM (Ubuntu).
* **HTML5/CSS3**: Interface moderna, limpa e responsiva focada em legibilidade.

## 📋 Funcionalidades

* **Arquitetura em Container**: Deploy rápido e consistente em qualquer ambiente com suporte a Docker.
* **Foco em Infraestrutura**: Destaque para competências em Redes (Fortigate, VPN), Observabilidade (Zabbix, Grafana) e Segurança.
* **Alta Disponibilidade**: Configuração otimizada para garantir que o serviço web responda prontamente na porta 8080.
